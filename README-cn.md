👉 [**API文档**](https://shenmian.github.io/Clementine) (中文)  

## 特点 🎄
- 多平台: Linux, macOS, Windows
- 轻量

## 编译 ⛓

### 构建脚本
``` bash
> git clone https://github.com/ShenMian/Clementine.git
> cd Clementine
> ./build.bat    # Windows
> ./build.sh     # Linux, MacOS
```

### CMake 选项 (默认 ON)
- `BUILD_EXAMPLES` 构建示例
- `BUILD_TESTS`    构建单元测试
