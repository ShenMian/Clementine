Clementine
A framework for building 2D console games.
README-cn

zh | en


这是一个用于制作2D命令行游戏的框架.

👉 API文档 (中文)

特点 🎄

  • 多平台: Linux, macOS, Windows
  • 轻量

编译 ⛓

构建脚本

> git clone https://github.com/ShenMian/Clementine.git
> cd Clementine
> ./build.bat # Windows
> ./build.sh # Linux, MacOS

CMake 选项 (默认 ON)

  • BUILD_EXAMPLES 构建示例
  • BUILD_TESTS 构建单元测试
